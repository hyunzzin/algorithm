## 뒤집기
1을 0으로 0을 1로 연속된 하나 이상의 숫자를 잡고 모두 뒤집기

</br>

### 접근법
1. 문자가 몇번 바뀌는지 세기
2. 0과 1이 n-1번, n번이 될테니 바뀌는 횟수를  2로 나눠주기

</br>

### 구현

```python
import sys
S = list(sys.stdin.readline().strip('\n'))
count = 0
pre = S[0]
for s in S:
    if pre != s:
        count += 1
        pre = s
if count%2:
    print((count+1)//2)
else:
    print(count//2)
```
